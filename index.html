Springwithdojo

This is a open source project that demos how to use Spring with Dojo

This is a sample project I made to show how to use Maven, Spring, Webflow and Dojo. You will find a number of examples that can help you

Getting The Project and Running It

To get this project and run it you will need to follow the following steps:

git clone git@github.com:JohnathanMarkSmith/SpringWithDojo.git
cd SpringWithDojo/
mvn tomcat7:run

Now open your web brower and goto http://127.0.0.1:8080/SpringWithDojo/
